XSF Source Control

All specifications, XML schemas, and registries produced by the XMPP Standards Foundation are kept under source control using the open-source Subversion system. Anonymous checkout of these files can be completed via the following Subversion command:

svn co svn://svn.xmpp.org:7938/xmpp .

The repository can be viewed and tracked in the following ways:

  • Git mirror: <http://github.com/xsf/documentation>
  • Commits list: xmpp-commits@xmpp.org
  • RSS feed: rss.xml

If you are a XEP author or a member of the XSF's Communication Team or Technical Review Team, please send your RSA/DSA key to Peter Saint-Andre for secure check-in privileges; instead of what is shown above you will use the following command for checkout:

svn co svn+ssh://svn@svn.xmpp.org/xmpp .
