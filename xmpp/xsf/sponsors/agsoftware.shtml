XSF Sponsors :: AG Software

AG Software, a Silver Sponsor of the XMPP Standards Foundation (XSF), is an independent software developer offering code development and consulting related to advanced communications technologies.

AG Software's products include SmartJab, a Jabber/XMPP client for Microsoft Mobile based Smartphones, and agsXMPP, an open-source library for Jabber/XMPP development based on Microsoft .NET Technology.

For more information about AG Software, visit <www.ag-software.de>.
