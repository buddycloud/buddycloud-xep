XSF Sponsors :: Jive Software

Jive Software builds next-generation collaboration solutions. Jive Software is the developer of the Openfire server, Spark client, and Smack Java library, three leading professional Open Source projects for EIM, presence, and XMPP-based communications. Built using pure Java, Openfire is a scalable, secure, cross-platform XMPP server that can be easily and quickly customized using simple plugins. It installs in a matter of minutes and provides enterprise-class features such as message auditing, archiving and LDAP integration. An enterprise edition of Openfire is also available under a commercial license. Openfire Enterprise is implemented as a standard plugin and provides an expanded set of features such as real-time reporting, advanced archiving and granular client control.

Spark, a full-featured XMPP client, provides an exceptional IM user experience and is tightly integrated with the Openfire server. Spark's plugin architecture allows you to buy or build plugins (Sparkplugs™) that add new application features tailored for your business. Spark is centrally managed from within the Openfire administration console, making it easy to provision and update all clients on your network.

Jive Software has over 1,350 customers world-wide, including IBM, Sun, Oracle, BEA, Deutsche Telekom, USJFCOM, Citrix and Toshiba.
