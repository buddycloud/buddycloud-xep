XMPP Interoperability Testing Network

The XMPP Standards Foundation coordinates a private network of XMPP servers that developers can access in order to test their code against a wide range of client and server implementations. This "real-time testing lab" helps the XMPP developer community ensure interoperability among implementations. The following pages contain detailed information.
