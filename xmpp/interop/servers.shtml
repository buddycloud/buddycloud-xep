XMPP Interop :: Servers

The core of the XMPP interop network consists of one server deployment for each XMPP server implementation. So far the following servers are online, with more to come:

Address Implementation Host Contact Person
coversant.interop.xmpp.org SoapBox Server Coversant Chris Mullins
djabberd.interop.xmpp.org djabberd Six Apart Brad Fitzpatrick
djabberd-trunk.interop.xmpp.org djabberd Six Apart Brad Fitzpatrick
ejabberd.interop.xmpp.org ejabberd Process-one Jérôme Sautret
openfire.interop.xmpp.org Openfire Jive Software Gaston Dombiak

To obtain an account on one of these servers, ask the contact person for the server, who will verify that you are an authorized participant and then give you a username and password.

To add a server to this list, contact Peter Saint-Andre, who will point the appropriate DNS entry at your IP address.
