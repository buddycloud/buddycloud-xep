XMPP Software

The Extensible Messaging and Presence Protocol (XMPP) has been implemented in dozens of clients, servers, and code libraries, some of them going all the way back to the first open-source Jabber projects in 1999. The following pages provide lists of XMPP implementations, without any warranty that the software is fully compliant with the XMPP specifications.
