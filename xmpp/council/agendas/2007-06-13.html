XMPP Council Meeting Agenda, 2007-06-13

  1. Roll call.

  2. Schedule next meeting (June 27?).

  3. XEP-0080: User Geolocation

    Approve version 1.4pre2?

    Changelog: Renamed to User Location; corrected pubsub examples; removed non-pubsub examples; added uri element.

  4. XEP-0106: JID Escaping

    Approve version 1.1pre2?

    Changelog: Specified that \20 must not be included at the beginning or end of a node identifier; added note about native JIDs with escaped characters; added mapping for IRC addresses; modified terminology to consistently use escaping and unescaping rather than encoding and decoding.

  5. XEP-0060: Publish-Subscribe

    Approve version 1.10pre1?

    Changelog: In accordance with XMPP Council consensus, moved the auto-create, auto-subscribe, filtered-notifications, and last-published features from XEP-0163 to this specification and defined them more precisely; also defined one disco#info feature for each default access model.

  6. XEP-0163: PEP

    Approve version 1.1pre4?

    Changelog: In accordance with XMPP Council consensus (1) explicitly defined auto-create, auto-subscribe, filtered-notifications, and last-published features, (2) moved them to XEP-0060, and (3) added appropriate references to XEP-0060 throughout; also added friendly but non-normative How It Works section and removed references to private data storage.

  7. XEP-0211 and XEP-0212

    Acceptable to refer to rfc3920bis and rfc3921bis rather than RFC 3920 and RFC 3921?
