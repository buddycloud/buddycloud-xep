  11. XEP-0045: Multi-User Chat

    Approve version 1.23pre6?

    Changelog: Defined getmemberlist room configuration option; added direct invitation protoocol; corrected logic regarding admission of room owner/admin when room is full; defined service discovery extension field for associated LDAP group; specified that room config fields can be listed in extended room information; specified message format for affiliation change notifications if user is not in the room; added example showing use of Result Set Management; recommended inclusion of MUC child element in presence errors; described use of ThreadID for continuity between one-to-one chat and multi-user chat, including definition of thread attribute for continue element in invitations.

    Diff

    Votes needed from all Council members.

  12. XEP-0115: Entity Capabilities

    Approve version 1.5pre12?

    Changelog: Specified that inclusion of hash attribute is required, removed default value of sha-1, and added reference to IANA registry; mentioned pre-image attack and added reference to RFC 4270; clarified meaning and construction of caps node attribute and disco node attribute; specified that node attribute shall be included in disco#info request for backwards-compatibility; further specified security considerations; clarified handling of the legacy format to assist developers; defined optional v attribute for the software version; added service discovery feature for caps optimization to prevent confusion regarding server support of caps vs. caps optimization.

    Diff

    Votes needed from all Council members.

  13. XEP-0155: Stanza Session Negotiation

    Approve version 1.2pre2?

    Diff

    Votes needed from all Council members (all members but Ian previously voted, but the specification has changed since then).
